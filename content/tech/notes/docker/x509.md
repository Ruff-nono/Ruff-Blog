## 问题

在进行 `docker login` 时，报错x509错误
![11](error.png)

检查私有仓库配置
```shell
sudo vi /etc/docker/daemon.json
```
``` config
{
        "insecure-registries": ["私有仓库地址"]
}
```

根据系统选择合适的命令重启docker服务
```shell
sudo systemctl restart docker 
# init
sudo service docker restart
```
