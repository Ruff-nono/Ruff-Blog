[Unity Hub下载](https://unity.com/cn/download)

[Vs Code下载](https://code.visualstudio.com/download)

1. 根据系统选择对应安装包，分别下载安装Unity Hub、Vs Code 
2. 打开 Unity Hub，进行登录 
3. 点击左侧 Install，选择 Unity 版本进行安装 
4. 打开 Vs Code, 配置VsCode插件，选择左侧 Extensions(Ctrl+Shift+X)，搜索安装`C#`、`Unity Code Snippets`
